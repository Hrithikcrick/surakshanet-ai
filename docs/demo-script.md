# SurakshaNet AI — Five-Minute Demo Script

## 0:00–0:30 — Problem

Digital-arrest scams and authority-impersonation attacks use fear, urgency, secrecy and payment pressure.

Existing tools often analyse only a suspicious message or only a financial transaction.

SurakshaNet AI combines communication intelligence, transaction-risk analysis and automated incident response.

## 0:30–1:00 — Solution

SurakshaNet follows five stages:

1. Detect
2. Understand
3. Score
4. Act
5. Record

The system combines:

- Deterministic fraud rules
- Google Gemini contextual analysis
- Transaction anomaly detection
- Explainable hybrid scoring
- n8n automation
- Telegram alerts
- Investigator case storage

## 1:00–2:00 — Live Fraud Analysis

Open the Fraud Analysis page.

Use the digital-arrest sample:

This is CBI. Your Aadhaar is linked to money laundering. Join a video call immediately or you will be digitally arrested. Transfer money for verification.

Click Analyze Fraud Risk.

Show:

- Rule Engine score
- Gemini LLM score
- Transaction Engine score
- Final Hybrid Risk Score
- Gemini scam type
- Gemini confidence
- Evidence
- Manipulation tactics
- Recommended action

Say:

“SurakshaNet does not provide only a black-box AI answer. It explains how every component contributed to the final score.”

## 2:00–2:40 — Automated Alert

Open Telegram.

Show the automatically generated alert containing:

- Case ID
- Victim ID
- Final risk score
- Risk level
- Priority
- Scam verdict
- Transaction decision
- Investigation summary
- Recommended next steps

Say:

“The analysis immediately becomes an operational response through n8n.”
