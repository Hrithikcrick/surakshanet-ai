# Hackathon Submission Summary

## Project Name

SurakshaNet AI

## Tagline

Hybrid LLM Fraud Intelligence and Automated Response Platform

## Problem Statement

Digital-arrest scams, authority impersonation, phishing, credential theft and payment coercion are becoming increasingly sophisticated.

Victims and investigators need a system that can analyse suspicious communication and transaction behaviour together, explain the risk, trigger an immediate response and preserve an investigation record.

## Solution Overview

SurakshaNet AI combines:

- Deterministic fraud rules
- Google Gemini contextual intelligence
- Transaction anomaly detection
- Explainable hybrid risk scoring
- SQLite investigation records
- n8n workflow automation
- Telegram incident alerts
- Complaint-support package generation

## Innovation

SurakshaNet is not simply an LLM chatbot or keyword detector.

It combines:

Deterministic reliability
+
LLM contextual understanding
+
Financial-risk signals
+
Automated response
+
Investigation documentation

## Evaluation

Gemini was evaluated using a balanced sample from the UCI SMS Spam Collection.

Results:

- Accuracy: 90%
- Precision: 90%
- Recall: 90%
- F1 score: 90%

The sample contained:

- 10 spam messages
- 10 legitimate messages

The deterministic rule engine is specialised for high-risk social-engineering scams and is not presented as a generic spam classifier.

- Graph-based scam-network analysis
- Investigator authentication

## Responsible AI

SurakshaNet is an investigation-support system.

It must not be used as the sole basis for:

- Accusing an individual
- Blocking an account permanently
- Filing a criminal allegation
- Taking legal action
- Making a final financial decision

All high-impact decisions require human review.
